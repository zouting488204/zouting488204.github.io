promise

概述：promise在异步执行的流程中，把执行代码和处理结果的代码分离了

简单的promise例子：

var pfn = function (b){
return new Promise(function(resolve,reject){
if(b>3){
resolve(‘大于3’)
}else{
reject(‘小于3’)
}
})
}

pfn(5).then(function(){
    console.log('成功')
}).catch(function(){
    console.log('失败')
})

先做任务1，任务1成功之后再执行任务2

job1.then(job2).then(job3).catch(handleError);

任务并发执行：

promise.all([p1,p2]).then(function(resolve){
})
