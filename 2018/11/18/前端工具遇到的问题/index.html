前端工具遇到的问题

1、window SourceTree 免登录

原文：https://blog.csdn.net/chenqiai0/article/details/83472410 

我从官网上下载了信息的3.0.8版本，注册的时候发现需要进行注册。由于没有账号，于是在Google搜索跳过登陆，
但是一直没有成功。 想到是不是最新版本修复了这个漏洞，于是就下载了一个低版本：2.6.10.0
https://www.sourcetreeapp.com/download-archives 
然后再按照下述博客的方法即可：https://www.cnblogs.com/xiofee/p/sourcetree_pass_initialization_setup.html
新建一个accounts.json 保存在\LocalAppData\Atlassian\SourceTree\下即可
[
{
  "$id": "1",
  "$type": "SourceTree.Api.Host.Identity.Model.IdentityAccount, SourceTree.Api.Host.Identity",
  "Authenticate": true,
  "HostInstance": {
    "$id": "2",
    "$type": "SourceTree.Host.Atlassianaccount.AtlassianAccountInstance, SourceTree.Host.AtlassianAccount",
    "Host": {
      "$id": "3",
      "$type": "SourceTree.Host.Atlassianaccount.AtlassianAccountHost, SourceTree.Host.AtlassianAccount",
      "Id": "atlassian account"
    },
    "BaseUrl": "https://id.atlassian.com/"
  },
  "Credentials": {
    "$id": "4",
    "$type": "SourceTree.Model.BasicAuthCredentials, SourceTree.Api.Account",
    "Username": "",
    "Email": null
  },
  "IsDefault": false
}
]

2、解决Mac下SourceTree每次都让输入密码的问题

原文：https://blog.csdn.net/aiqinxuancai/article/details/69817496
英文： SourceTree -> Preferences -> Git -> Use System Git
中文版：SourceTree -> 偏好设置-> Git -> 使用系统安装的Git

3、mac破解ps,显示没有framework 文件怎么办

原文：https://zhidao.baidu.com/question/585785572002863605.html
打开Finder-应用程序，然后找到PS，双击打开PS，然后进入一个文件夹，还会看到一个PS文件，右击-显示包内容，就可以看到了。

5、window jre1.8 安装并使用

Windows x64 71.16 MB    jre-8u191-windows-x64.exe   直接下载exe格式的，点击下一步安装即可

5、window7 取消启动项：

1、按win+R键打开运行，输入“msconfig”，点击确定
2、点击启动
